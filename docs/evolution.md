## V1 — 灵魂注入（约 2026-06-09 21:00）
**"让分身有记忆"**

```python
# 第一版：能跑就行
agent = AIAgent(skip_memory=False, ...)
agent.run_conversation(prompt)
```

- ✅ 第一个能推理的 Hermes 分身诞生
- ✅ 继承主记忆（知道军哥是谁）
- ❌ 回调签名不对 → crash
- ❌ 产出只能 stdout 抓

**军哥的核心设计**：时间线分支模型。时间线1 聊天不中断，时间线2 独立推理，完成后记忆 merge 回时间线1。

## V6 — 原地等待（约 23:30）
**"101 不退出，等军哥回复后从断点续跑"**

```
while True:
    run_agent()
    if 无求助: break
    wait_for_answers()  # 轮询 /tmp/tentacle_help/answered_*.json
    continue            # 续跑！
```

- ✅ 101 不再退出 → 原地轮询（3 秒检查一次）
- ✅ 主循环支持多轮续跑
- ✅ 小雪不做风险判断 → 一律报军哥
- ✅ 支持多轮求助（求助→等→续跑→求助→等→续跑...）

## V6-final — 五处加固（约 00:00）
**"自己审视自己，把漏的全补上"**

| # | 问题 | 修复 |
|---|------|------|
| ① | 等待循环没心跳 → 看门狗误判死循环 | 等待时也写 heartbeat |
| ② | 3600s 等待超时 → 等不到就放弃 | 去掉超时，等到回复或被 kill |
| ③ | max_iterations=30 硬编码 → 大任务截断 | 改为 `--max-iterations`，默认 50 |
| ④ | 无 SIGTERM → kill 时产出一滴不剩 | signal 处理 → 保存部分结果 |
| ⑤ | heartbeat 无异常保护 → 磁盘满就崩 | try/except 静默吞掉 |
