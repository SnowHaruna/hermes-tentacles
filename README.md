# 🐙 Hermes' Tentacles

> **从对话中自动分身。** AI 判断任务太重 → fork 完整克隆到后台 → 独立推理 → 记忆自动回传。你全程没等过一秒。

对话驱动的 AI Agent 动态分身系统。Auto-triage → Fork → 独立推理 → 记忆回传。

## 为什么

现有的 AI 助手做重活只有三条路：

| 方式 | 问题 |
|------|------|
| 前台干活 | 你干等 |
| 后台脚本 | 只会跑命令，不能推理 |
| 子 Agent | 干活时冻结聊天 |

**触手是第四条路。**

## 怎么工作

```
用户一句话
     │
     ▼
  auto-triage
  ╱        ╲
轻任务      重任务
(秒答)      (自动 fork 🦑)
              │
         独立 Hermes 实例
         ├─ 继承主记忆
         ├─ 搜索 / 分析 / 推理
         ├─ 写报告到文件
         ├─ 发现写回记忆池  ← 独创
         └─ 汇报主 Hermes
```
